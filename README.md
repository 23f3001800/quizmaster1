# 🎯 QuizMaster1  
### AppDev1 Project – IITM BS Degree (IIT Madras BS in Data Science)

QuizMaster1 is a full-stack web-based quiz application built using **Flask** for the backend and **Jinja2** for frontend templating.

This project was developed as part of the **Application Development 1 (AppDev1)** course in the IIT Madras BS Degree Program.  
The application is fully functional, tested locally, and deployed to GitHub for showcasing.

---

## 🌐 Project Overview

QuizMaster1 is designed to demonstrate core web development concepts including:

- Backend development using Flask
- Dynamic template rendering with Jinja2
- User authentication and session handling
- Database integration
- MVC-based application structuring
- CRUD operations for quizzes and questions

The application allows users to register, log in, attempt quizzes, and view their scores.

---

## 🚀 Features

- 👤 User Registration & Login System  
- 🔐 Secure Session-Based Authentication  
- 📝 Quiz Creation and Management  
- ❓ Add, Edit, and Delete Questions  
- 📊 Score Calculation and Display  
- 🎨 Clean UI rendered with Jinja2 Templates
